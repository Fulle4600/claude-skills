# /compact — Sammenfat samtalen til ny chat

Sammenfat hele vores samtale så den kan indsættes i en ny chat uden at miste kontekst.

Inkludér:
1. Det oprindelige mål eller problem
2. Vigtige beslutninger truffet og hvorfor
3. Al kode, config eller data vi er nået frem til — verbatim i kodeblokke
4. Åbne spørgsmål og næste skridt

Brug korte sektioner med overskrifter. Spring small talk og tangenter over. Optimer sammenfattet til en fremtidig Claude der læser det koldt uden forudgående kontekst.
